人工智能算法工程师

要求：

有一定的机器人运动学和人工智能知识，一定的编程功底，有实际项目经验者优先，有RoboCup等重要机器人比赛参赛经验者优先

具有以下一项或多项能力：

  1. 人工智能算法开发

  2. 理解掌握机器人学主要理论，有运动学、人工智能等综合开发经验

视觉算法程师

要求：

有一定机器视觉理论知识和实际开发经验，熟练掌握C和Python编程

具有以下一项或多项能力：

  1. 计算机视觉相关算法的开发，人脸识别和人形识别算法的研究

  2. 调研使用现有算法API接口的设计、集成和优化

  3. 理解SLAM算法并能够进行实际开发

  4. 熟悉OpenCV等开源库

嵌入式开发工程师

要求：

  1. 负责对各个电机进行位置、速度控制

  2. 设计开发电池管理模块

  3. 熟悉常用通讯接口，如USB口、并口、串口、SPI、IIC等

  4. 负责WiFi通信模块、摄像头驱动模块的调试和使用

  5. 丰富硬件设计和调试经验，有基本电机调试知识和经验，有能力学习和使用购买到的全新电机和各种开发模块。有PCB开发经验者优先

安卓开发工程师

要求：

  1. 安卓前端程序逻辑功能的开发

  2. 熟练掌握常用android API ，深入理解android 平台架构、主要组成和工作模式

  3. 有丰富手机软件开发经验，熟练掌握Python、Java、C#等编程语言

  4. 熟悉TCP/IP等网络协议原理

  5. 了解openfire framework，xmpp协议者优先

IT工程师

要求：

  1. 参与公司服务器搭建与维护、git维护、网络及IT技术支持服务

  2. 熟悉各类计算机操作系统安装、硬件软件维护

  3. 独立完成网络的设备配置与管理及快速排错能力

  4. 熟练使用Linux系统，熟练掌握Python、Java等编程语言

造型设计师

要求：

  1. 参与产品造型设计、手工制造、CAD建模，与机械设计师交流选择合理的硬件设计

  2. 熟悉各类计算机辅助设计软件，如Rhino, Alias, Maya, vray, CAD, PS, Ai等

  3. 有较强的美术功底，热爱设计

  4. 有一定的工业设计和团队协作经验

  5. 手绘表现及动手制作能力强，有一定的建模和渲染能力

  6. 有一定编程能力者优先
